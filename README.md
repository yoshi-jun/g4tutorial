# g4tutorial

You must import config file to run this program 
~~~
./Application_Main -c config.json
~~~
config.json
~~~
{
  "beam_flag" : 0,
  "save_flag" : 1,
  "dim_x"    : 61,
  "dim_y"    : 61,
  "dim_z"    : 150
}
~~~

beam_flag  
- 1 broad beam
- 0 pencil beam 

save_flag  
- 1 save to file 
- 0 save to test file

dim_x,y,z
- numeric of boxcell
